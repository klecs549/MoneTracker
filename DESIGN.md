# Design System

This project uses a light theme with red accent

---

## Stack

- **Framework:** Vite + React + TypeScript
- **Styling:** CSS for each componnent/page
- **Components:** shadcn-style primitives in `components`
- **Icons:** Lucide React
- **Fonts:** IBM Plex Sans for UI, Fraunces for display headings


---

## Visual Direction

The app is a Moneytracker

- Use light ink backgrounds, warm paper overlays, amber command accents, teal graph highlights.
